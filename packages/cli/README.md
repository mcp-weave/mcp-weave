# @mcp-weave/cli

CLI tool for MCP-Weave - transform your code into MCP servers using decorators.

## Installation

Via npm:

```bash
npm install -g @mcp-weave/cli
```

Via pnpm:

```bash
pnpm add -g @mcp-weave/cli
```

## Usage

After installing globally, use the command:

```bash
mcp-weave [options]
```

### Commands

```bash
mcp-weave init              # Initialize a new MCP project
mcp-weave generate          # Generate server code from mcp-spec.yaml
mcp-weave extract           # Extract spec from decorated code
mcp-weave start             # Start the MCP server
```

For help:

```bash
mcp-weave --help
```

## Requirements

- Node.js >= 18
